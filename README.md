# Ankerのプロジェクター 投影サイズシミュレーター

プロフェッショナル仕様のAnkerプロジェクター投影シミュレーションツール。購買決定を支援する直感的でリッチな体験を提供します。

## 🌟 主要機能

### 1. **デュアルビューシミュレーション**
- **📐 上から見た図**: 投影距離と画面サイズの関係を光の広がりとともに可視化
- **🖼️ 正面から見た図**: 壁（高さ2.4m）との比較で実際のサイズ感を体験
  - ドア（2.0m）のシルエット表示
  - 天井との位置関係を確認
  - 大画面時の圧倒的なサイズ感を実感

### 2. **動的な明るさシミュレーション**
- プロジェクターの明るさ（ANSIルーメン）に応じて背景の明るさを変化
- 3500ルーメン: 明るい背景でも鮮明
- 1800ルーメン: 適度な明るさ
- 300ルーメン以下: 暗い背景推奨

### 3. **インテリジェントな逆引き計算**
- **希望サイズ → 必要距離**: 「100インチがほしい」→「2.7m必要」
- **距離 → 画面サイズ**: 「3m離せる」→「110インチ可能」
- リアルタイムスライダーでヌルヌル動く体験

### 4. **プロ仕様の寸法表示**
- 矢印付き寸法線（Dimension line）
- 半透明バッジで視認性確保
- 「投影距離: 2.70m」のように項目名付き
- フォントサイズ大型化（最大22px）で読みやすく

### 5. **スマートなエラーハンドリング**
- 最大投影サイズ超過時に即座にアラート
- 図の中に赤い×印と「投影不可」表示
- 対応範囲を明示して代替案を提示

### 6. **リアルタイムフィードバック**
- スライダーツールチップで現在値を表示
- 部屋の広さアドバイス（必要な畳数を計算）
- 明るさに応じた使用環境のアドバイス
- 購入ボタンで公式サイトへ直リンク

### 7. **モデル比較機能**
- 2機種を並べて詳細比較
- スペック一覧表示（解像度、明るさ、重量、バッテリーなど）
- 投影範囲の違いを一目で確認

## 🚀 使い方

1. **計算モードを選択**
   - 📐 希望サイズから距離を計算（デフォルト）
   - 📏 距離からサイズを計算

2. **値を入力**
   - スライダーまたは数値入力
   - リアルタイムでツールチップ表示

3. **プロジェクターを選択**
   - 製品画像付きカードをクリック

4. **ビューを切り替え**
   - 📐 上から見た図：投影の仕組みを理解
   - 🖼️ 正面から見た図：実際のサイズ感を体験

5. **結果を確認**
   - 画面サイズ（インチ・cm）
   - 必要な部屋の広さ
   - 明るさアドバイス
   - 購入リンク

## 🎨 技術仕様

- **Pure Vanilla JavaScript**: ビルド不要、ブラウザで即動作
- **SVG グラフィックス**: ベクターベースで美しく拡大
- **CSS アニメーション**: 60fps の滑らかな動き
- **レスポンシブデザイン**: モバイルからデスクトップまで対応
- **リアルタイム計算**: 線形補間による正確なシミュレーション
